# Catppuccin Mocha (Zed) for VS Code

A VS Code theme that matches the Catppuccin Mocha theme from Zed editor.

## Installation

### From VS Code Marketplace
Search for "Catppuccin Mocha (Zed)" in the Extensions view.

### Manual Installation
1. Clone this repository
2. Copy the folder to ~/.vscode/extensions/
3. Restart VS Code
4. Select the theme: Cmd+K Cmd+T -> Catppuccin Mocha (Zed)

## Recommended Settings

For the complete Zed experience, add these to your VS Code settings.json:

```json
{
  "editor.fontFamily": "Lilex",
  "editor.fontSize": 14,
  "editor.fontLigatures": true,
  "editor.lineHeight": 1.6
}
```

**Note:** You will need to install [Lilex](https://github.com/mishamyrt/Lilex) font first. This is the default monospace font used by Zed editor.

## Features
- Exact syntax highlighting colors from Zeds Catppuccin Mocha
- Matching UI colors (sidebar, tabs, status bar, etc.)
- Dimmed whites for reduced eye strain
- Clean borders between panels

## Color Palette
- Keywords: Mauve (#cba6f7)
- Functions: Blue (#89b4fa)
- Strings: Green (#a6e3a1)
- Classes/Types: Orange (#fab387)
- Constants/Numbers: Yellow (#f9e2af)
- Comments: Gray (#9399b2)
- Operators: Sky (#89dceb)
